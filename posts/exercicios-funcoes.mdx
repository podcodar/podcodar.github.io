# Exercício de Programação: explorando funções

## Introdução

Funções são estruturas das linguagens de programação que permitem o **encapusalamento** de **abstrações** e **conceitos**. Quando criamos uma função, o trecho de código que ela encapsula recebe um identificador, comumente iniciado por um verbo, pois estamos descrevendo um conjunto de ações que será executado pelo computador.

Ex:
```python
# soma dois números, x e y, e retorna o resultado da soma
def somar(x, y):
    return x + y
```
- *def* e *return* são palavras **reservadas** da linguagem. 
    + *def* é utilizado para definir uma função
    + *return* é utilizado para retornar um valor
- *somar* é o identificador da função
- *x* e *y* são os **argumentos** que a função espera. No caso, os operadores a serem somados

O identificador dado a função **documenta** a abstração implementada e permite que ela seja referenciada em outros pontos do código. Assim, ao invés de repetir um mesmo trecho de código inúmeras vezes ao longo de uma aplicação, podemos **reaproveitá-lo** onde for necessário **chamando** a função pelo nome. Por exemplo, ao invés de re-implementar o código para verificar se um número é positivo toda vez que tal verificação for necessária, podemos criar uma função que encapsula este conceito e chamá-la informando qual o número que deve ser verificado.

Ex:
```python
def verificarNumeroPositivo(numero):
    if (numero > 0):
        return True
    else:
        return False

verificarNumeroPositivo(10)
verificarNumeroPositivo(0)
verificarNumeroPositivo(-1)
```

## Exercício 3: Interagindo com o usuário

Existem funções que já são definidas pela linguagem de programação. Elas abstraem ações comumente utilizadas nos programas.
Nos exercícios anteriores fizemos uso de uma dessas funções, a *print*, que é utilizada para imprimir textos na tela. Pense um pouco sobre a complexidade que esta função abstrai. Muito deve estar sendo feito por debaixo dos panos, mas, como **usuários da função**, não precisamos nos preocupar com estes detalhes.

Outra função que podemos utilizar para interagir com o usuário é a *input*. Ela interrompe o fluxo do programa a espera de uma entrada do usuário. Esta entrada pode ser um número ou texto.

Até então, a nossa calculadora apenas opera números definidos estaticamente no escopo do código. Vamos adicionar mais flexibilidade à nossa aplicação?

Utilizando a função *input*, pergunte ao usuário os operandos de cada operação e imprima o resultado a seguir.

Ex:
```python
def somar(x, y):
    return x + y

print('*** Soma ***')
operando1 = int(input('Digite o primeiro operando: '))
operando2 = int(input('Digite o segundo operando: '))
resultadoSoma = somar(operando1, operando2)
print('Resultado soma:', resultadoSoma)
```

Note que estamos **aninhando** funções passando o **retorno** da função *input* para a função *int*. Isso é necessário porque a função *input* retorna um valor do tipo *string*, então precisamos convertê-lo em um valor do tipo *inteiro*.
