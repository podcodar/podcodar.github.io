## Introdução
Dando continuidade a nossa jornada, aprendemos o tipo de dados *list*. Listas são sequências de items de tipo qualquer. Podemos adicionar e remover items da lista, verificar a existência de um elemento e iterar sob a lista. Veja alguns exemplos abaixo:

```
>>> mylist = [20, 'lua', 100, 1.5]
>>> print(mylist)
[20, 'lua', 100, 1.5]
>>> mylist.append("um novo item")
>>> print(mylist)
[20, 'lua', 100, 1.5, 'um novo item']
>>> mylist.remove(1.5)
>>> print(mylist)
[20, 'lua', 100, 'um novo item']
>>> print('lua' in mylist)
True
>>> print('sol' in mylist)
False
>>> for item in mylist:
...   print(item)
20
lua
100
um novo item
```
