## Exercícios - Agenda PodCodar
A PodCodar está crescendo e precisa da sua ajuda para fazer um programa que gerencie as informações dos alunos.
Gostaríamos de poder adicionar novos alunos, consultar as informações de um aluno, consultar presença e entrega das listas de exercícios, entre outras funcionalidades.

O tipo dicionário que acabamos de aprender é uma boa estrutura para guardarmos os dados dos alunos. A chave deverá ser o *alias* do aluno e o valor um outro dicionário com as informações do aluno. Veja o exemplo a seguir:
```python
# A agenda da PodCodar será definida como um dicionário tal que:
# - a chave é um alias que identifica a pessoa
# - o valor é um dicionário com todas as informações da pessoa
podcodar_notebook = {
    'b1': {
        'full_name': 'Banana B1',
        'city': 'Belo Horizonte',
        'dob': '01/01/1990', # dob means date of birth
        'email': 'b1@hotmail.com',
        'buddy': 'Luiz'
    },
    'b2': {
        'full_name': 'Banana B2',
        'city': 'São Paulo',
        'dob': '01/01/1990',
        'email': 'b2@hotmail.com',
        'buddy': 'Marquin'
    },
    'b3': {
        'full_name': 'Banana B3',
        'dob': '01/01/1992',
        'email': 'b2@hotmail.com',
    },
}
```
